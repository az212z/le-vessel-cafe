# DESIGN-QUALITY-REPORT — لي فيسل · LE VESSEL CAFE

## 1. المهارات المُستدعاة وكيف طُبّقت

### `ui-ux-pro-max` (+ design-system search)
شغّلت: `search.py "specialty coffee cafe modern" --design-system`.
- المخرج اقترح نمط **Hero-Centric + Conversion** ولوحة بنّي/كريمي دافئة وخطوط Outfit/Work Sans.
- **القرار:** اعتمدت بنية Hero-Centric + Conversion (هيرو قوي ثم قيمة ثم CTA)، لكن **استبدلت اللوحة البنّية والخطوط اللاتينية** بتوجيه البريف المميّز (تيل/كحلي + كريمي + نحاسي، وخطوط عربية أنيقة) — لأن البنّي هو الـ default المتوقّع لكل كوفي، والبريف طلب هوية **مميّزة** تمنع التشابه.
- طبّقت من القائمة المرجعية: لا emoji كأيقونات (كل الأيقونات SVG inline)، تباين ≥4.5:1، focus-visible، احترام prefers-reduced-motion، responsive على 375/768/1024/1440، حركة 150–300ms.

### `design-taste-frontend`
- **Design Read** صريح: premium-consumer landing لكوفي مختص، لغة nautical-minimal راقية.
- **الـ Dials:** VARIANCE 7 (هيرو غير متمركز، أعمدة غير متساوية في Why/Order)، MOTION 5 (reveal + hover + press)، DENSITY 3 (مساحات واسعة).
- منعت AI tells: لا em-dash إطلاقًا، لا بنفسجي AI، لا ثلاث كروت متطابقة (شبكة 6 خلايا فيها كرت accent مختلف)، لا locale/time strips، لا scroll cues، لا version labels، لا decorative dots، لا fake screenshots.
- **Color Consistency Lock:** لون النحاس accent واحد عبر كل الصفحة. **Shape Lock:** أزرار pill + كروت 14px ثابتة. **Theme Lock:** ثيم فاتح (كريمي) واحد مع أقسام تيل داكنة ضمن نفس العائلة.
- **استثناء palette ban:** اللوحة الدافئة هنا مبرّرة بالبريف صراحةً وهي تيل/كريمي/نحاس (ليست beige+brass+espresso الممنوعة) — وتمنع تكرار قالب الكوفي البنّي.
- صور حقيقية فقط (لا fake divs)، الهيرو بصورة حقيقية.

### `emil-design-eng`
- منحنيات easing مخصّصة: `--ease-out: cubic-bezier(0.23,1,0.32,1)` بدل الـ built-in الضعيف.
- **scale(0.97) on :active** لكل زر و FAB (إحساس الضغط الفيزيائي).
- لا حركة من `scale(0)` — الـ reveal من `translateY(24px)+opacity` (الأشياء لا تظهر من العدم).
- المدد ضمن النطاق: ضغط الزر 160ms، التحوّلات ≤600ms للريفيل، toast 250ms.
- transitions (لا keyframes) للعناصر التفاعلية لتكون قابلة للمقاطعة؛ نُحرّك transform/opacity فقط (GPU).

## 2. مخرجات design-system (palette/type) المعتمدة
- **palette:** `--teal #0E3A40`، `--cream #F5EFE3`، `--brass #B8863B`، `--ink #1C2326`. (متغيّرات CSS دلالية، لا hex مبعثر.)
- **typography:** Reem Kufi (عناوين) / Tajawal (جسم) / Cormorant Garamond (لاتيني "Le Vessel"). سلّم أحجام clamp متجاوب.

## 3. قرارات UI/UX الأساسية
- هيرو ملء الشاشة بصورة الواجهة الليلية + scrim متدرّج لضمان قراءة النص فوقها (تباين عالٍ).
- شريط ثقة يذكر تقييم قوقل الحقيقي (4.6 / 509).
- قائمة منتجات بشبكة 6 خلايا (5 فئات + كرت دعوة accent) بأيقونات SVG موحّدة.
- Why-us بقسم تيل داكن وقائمة بحدود نحاسية.
- معرض بأحجام بلاطات متنوّعة + lightbox خفيف.
- نموذج طلب → واتساب + localStorage + toast، موسوم بوضوح كتجريبي.
- FABs ثابتة (واتساب/اتصال/خرائط) + footer كامل.

## 4. سبب الألوان/الخطوط
التيل العميق + النحاس يعطيان إحساسًا بحريًا-فاخرًا هادئًا (يناسب اسم "Vessel" = وعاء/سفينة)، والكريمي يلطّف ويرفع التباين. الخطوط العربية الأنيقة + اللاتيني الكلاسيكي يطابقان طابع العلامة الراقي ويميّزانها عن قوالب الكوفي البنّية.

## 5. تطبيق Hooked / UX
- محفّز واضح (CTA "اطلب الآن") متكرر بنفس النية (لا تكرار نوايا متضاربة).
- فعل بسيط (نموذج قصير) → مكافأة (رسالة واتساب جاهزة + toast فوري).
- ردود فعل فورية على كل ضغط (<150ms).

## 6. تطبيق إرشادات الموبايل (HIG/Material)
- أهداف لمس ≥48px (أزرار، حقول، FABs، burger 46px).
- `min-h-dvh` لا `100vh`؛ safe spacing؛ بدون تمرير أفقي.
- قائمة جوال = **overlay ملء الشاشة 100vw/100dvh، خلفية تيل صلبة معتمة، زر X واضح** (تم التحقق حيًّا: width=innerW, bg=rgb(14,58,64)).

## 7. تطبيق Accessibility
- تباين: نص كريمي على تيل وداكن على كريمي — جميعها ≥4.5:1.
- HTML سيمانتيك (header/nav/main/section/footer)، تدرّج h1→h3 بلا قفز.
- alt عربي لكل صورة، aria-label لكل زر أيقونة، role/aria-modal للقائمة والـ lightbox.
- `:focus-visible` بإطار نحاسي 3px على كل عنصر تفاعلي.
- aria-live على الـ toast؛ أخطاء النموذج تحت الحقل + focus لأول حقل خاطئ.
- دعم prefers-reduced-motion (إيقاف الحركة والـ reveal).
- اللون ليس وحده ناقلًا للمعنى (أيقونات + نص في كل مكان).

## 8. تطبيق Impeccable / Taste (اختبار القبول §9)
- يبدو فاخرًا؟ نعم — لوحة هادئة، مساحات واسعة، حركة ناعمة، صور حقيقية أنيقة.
- سعودي مناسب للنشاط؟ نعم — نص محايد جندريًا، تقييم قوقل، واتساب، خرائط.
- يقنع خلال 3 ثوانٍ؟ نعم — هيرو واضح + قيمة + CTA فوق الطية.
- لا يشبه قالبًا مجانيًا؟ نعم — هوية بصرية مميّزة تبتعد عن البنّي النمطي.
- المسافات/الصور/الأزرار/الحركة متناسقة؟ نعم — نظام مسافات 4/8، radius موحّد، accent واحد.
