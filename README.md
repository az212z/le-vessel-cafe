# لي فيسل · LE VESSEL CAFE

موقع تعريفي فاخر (Arabic RTL) لكوفي مختص في الخبر الشمالية.

## المعلومات
- **الاسم:** LE VESSEL CAFE — لي فيسل
- **التصنيف:** كوفي مختص (Specialty Coffee)
- **المدينة:** الخبر الشمالية، المنطقة الشرقية
- **التقييم:** 4.6 على خرائط قوقل (509 تقييم) — مذكور في الموقع وفي JSON-LD
- **الجوال / واتساب:** 0567603326 / 966567603326

## لماذا اختير هذا النشاط
كوفي مختص بتقييم عالٍ (4.6 / 509) وصور حقيقية مميزة لواجهة ليلية أنيقة وأجواء راقية — مادة بصرية قوية تستحق موقعًا فاخرًا يبرز هويتها.

## فكرة التصميم — "الوعاء المصقول" (refined vessel)
- لوحة ألوان مميزة (مختلفة عن البنّي المعتاد للكوفي): **تيل/كحلي عميق + كريمي دافئ + لمسة نحاسية (Brass)**.
- إحساس هادئ، أنيق، بحري-مينيمال يناسب القهوة المختصة.
- خطوط: **Reem Kufi** للعناوين العربية، **Tajawal** للنص، و**Cormorant Garamond** للاتيني "Le Vessel".
- حركة ناعمة (scroll-reveal، scale-on-press، منحنيات ease-out مخصّصة).

## التقنية
موقع ثابت (vanilla) بدون أي build step: HTML + CSS + JavaScript خام فقط — جاهز لـ GitHub Pages.

## حالة الـ Backend
**غير مربوط بأي خادم.** نموذج الطلب تجريبي للعرض:
- يبني رسالة واتساب جاهزة على `wa.me/966567603326`.
- يحفظ الطلب محليًا في `localStorage` (عرض فقط).
- يظهر إشعار (toast) عند الإرسال. لا يوجد نظام دفع.

## مصدر الصور
صور حقيقية من ملف النشاط على خرائط قوقل، تمّت **مراجعتها يدويًا واختيار الأنسب** (استُبعدت صورة قائمة الأسعار لعدم عرض أسعار). الصور المستخدمة: واجهة وجلسات خارجية ليلية، ركن التحضير، مدخل ليلي، وكيك/حلى.
